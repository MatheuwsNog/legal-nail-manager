Política de Privacidade — Nail Manager

Última atualização: 1 de junho de 2026

Esta Política de Privacidade descreve como o aplicativo Nail Manager ("Aplicativo", "nós") coleta, usa, armazena e compartilha as informações pessoais das usuárias ("você", "Usuária") que utilizam nosso serviço. Estamos comprometidos com a proteção dos seus dados conforme a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018).

1. Quem somos

O Nail Manager é desenvolvido e operado por:

  • Responsável: Matheus Miranda Nogueira
  • E-mail de contato: matheuwsnog@gmail.com

Para fins de exercício de direitos previstos pela LGPD, dados adicionais de identificação (como documentos pessoais) podem ser solicitados via e-mail, mediante prévia autenticação do titular.

2. Informações que coletamos

Coletamos as seguintes informações quando você utiliza o Aplicativo:

2.1 Dados cadastrais

  • Nome completo
  • E-mail
  • Telefone
  • Data de nascimento
  • Nome comercial (do salão/espaço)
  • Senha (armazenada de forma criptografada via Firebase Authentication)

2.2 Dados de uso do serviço

  • Cadastro de clientes que você atende (nome, telefone, observações, data de nascimento)
  • Agendamentos realizados (datas, horários, procedimentos, valores)
  • Registros financeiros (receitas, despesas, métodos de pagamento das suas clientes)
  • Configurações do seu negócio (horários de expediente, mensagens automatizadas)

2.3 Dados de assinatura e pagamento

  • Plano contratado e status da assinatura
  • Histórico de transações
  • Dados de cartão de crédito NÃO são armazenados pelo Nail Manager. Eles são processados diretamente pelas plataformas Stripe e Mercado Pago, que possuem certificação PCI-DSS.

2.4 Dados técnicos

  • Identificador único do dispositivo
  • Token de autenticação Firebase (renovado automaticamente)
  • Logs de erros e crashes (via Sentry) — para diagnóstico e correção
  • Endereço IP e horário de acesso (para fins de segurança)

3. Como usamos seus dados

Utilizamos os dados para:

  • Fornecer e operar as funcionalidades do Aplicativo
  • Processar pagamentos e gerenciar sua assinatura
  • Enviar notificações sobre agendamentos, pagamentos e atualizações
  • Detectar e prevenir fraudes ou uso indevido
  • Melhorar o serviço com base em dados agregados e anonimizados
  • Cumprir obrigações legais e regulatórias

4. Base legal (LGPD)

O tratamento dos seus dados é fundamentado nas seguintes bases legais:

  • Execução de contrato (Art. 7º, V) — para prestação do serviço contratado
  • Consentimento (Art. 7º, I) — para notificações marketing (opcional)
  • Cumprimento de obrigação legal (Art. 7º, II) — registros fiscais e contábeis
  • Legítimo interesse (Art. 7º, IX) — segurança e prevenção de fraudes

5. Compartilhamento com terceiros

Compartilhamos seus dados estritamente com prestadores de serviço necessários ao funcionamento do Aplicativo:

  • Google Firebase (Google LLC, EUA) — autenticação e identificação de usuárias
  • Fly.io (EUA) — hospedagem do backend e banco de dados PostgreSQL
  • Stripe (Stripe, Inc., EUA) — processamento de pagamentos via cartão de crédito
  • Mercado Pago (MercadoPago.com Atividades de Internet Ltda., Brasil) — processamento de pagamentos via PIX e cartão
  • Amazon Web Services (AWS S3) (EUA) — armazenamento de backups criptografados
  • Sentry (Functional Software, Inc., EUA) — registro de erros técnicos

Não vendemos, alugamos ou cedemos seus dados a terceiros para fins de marketing ou propaganda.

6. Transferência internacional de dados

Alguns dos prestadores acima estão sediados fora do Brasil. Garantimos que essas transferências ocorrem mediante cláusulas contratuais padrão e/ou certificações reconhecidas internacionalmente, conforme exigido pela LGPD.

7. Seus direitos como titular

Conforme a LGPD, você tem os seguintes direitos:

  • Confirmação da existência de tratamento dos seus dados
  • Acesso aos seus dados
  • Correção de dados incompletos, inexatos ou desatualizados
  • Anonimização, bloqueio ou eliminação de dados desnecessários
  • Portabilidade dos dados a outro fornecedor (mediante requisição)
  • Eliminação dos dados tratados com seu consentimento
  • Informação sobre com quem compartilhamos seus dados
  • Revogação do consentimento a qualquer momento

Você pode exercer esses direitos:

  • Diretamente no Aplicativo, em Ajustes → Segurança → Excluir Minha Conta
  • Por e-mail: matheuwsnog@gmail.com

8. Retenção e exclusão dos dados

Mantemos seus dados enquanto sua conta estiver ativa. Quando você solicita exclusão da conta:

  • Dados pessoais identificáveis são removidos em até 30 dias
  • Dados financeiros (registros de pagamento) são mantidos por 5 anos para cumprimento de obrigações fiscais
  • Backups antigos contendo seus dados são purgados em até 90 dias

9. Segurança dos dados

Adotamos medidas técnicas e organizacionais para proteger seus dados:

  • Tráfego criptografado em todas as comunicações (HTTPS/TLS)
  • Senhas armazenadas com hash criptográfico (Firebase)
  • Autenticação via token JWT com expiração curta
  • Isolamento de dados entre usuárias (cada usuária só acessa seus próprios dados)
  • Backups automáticos diários armazenados de forma criptografada
  • Logs de auditoria de acessos administrativos

10. Armazenamento local no dispositivo

O Aplicativo armazena algumas informações no seu dispositivo para funcionar offline e de forma rápida:

  • Cache local dos seus dados (clientes, agendamentos, financeiro)
  • Token de autenticação
  • Preferências de uso (tema claro/escuro, etc.)

Esses dados ficam protegidos no armazenamento seguro do iOS/Android e são removidos ao desinstalar o Aplicativo ou fazer logout.

11. Adolescentes e capacidade civil

O Nail Manager é destinado a profissionais maiores de 16 anos. Adolescentes entre 16 e 18 anos podem utilizar o Aplicativo, mas devem ter ciência e, quando aplicável, assistência de seus pais ou responsáveis legais para fins de celebração de contratos de assinatura paga, conforme prevê o Código Civil brasileiro (Art. 4º).

Não coletamos intencionalmente dados de crianças menores de 16 anos. Caso identifiquemos cadastro em desacordo com esta política, reservamo-nos o direito de suspender ou excluir a conta.

12. Alterações nesta política

Podemos atualizar esta Política periodicamente. Mudanças significativas serão comunicadas através do Aplicativo. A data da última atualização sempre aparecerá no topo do documento.
